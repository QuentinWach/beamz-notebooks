# BEAMZ Notebooks

A curated collection of interactive Jupyter notebooks for learning about optics, photonics, and laser physics.

**Live site:** [quentinwach.github.io/beamz-notebooks](https://quentinwach.github.io/beamz-notebooks/)

## How Do I Contribute?

1. Add your `.ipynb` file to the `notebooks/` directory
2. Push to `main`
3. The site auto-deploys via GitHub Actions
